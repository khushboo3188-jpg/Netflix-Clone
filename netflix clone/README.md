# 🎬 Netflix Clone

A responsive Netflix landing page clone built using **HTML5** and **CSS3**.  
This project recreates the UI of Netflix's homepage with a modern design and responsive layout.

---

## 📸 Preview

![Netflix Clone Banner](./assets/bg.jpg)

---

## ✨ Features

- 🎥 Hero section with background image
- 📺 TV animation with embedded videos
- 📱 Responsive design using Flexbox and Media Queries
- ❓ FAQ section
- 🔗 Footer with multiple links
- 🎨 Modern Netflix-inspired UI
- 🌙 Dark theme

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- Media Queries
